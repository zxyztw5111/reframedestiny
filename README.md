## What is this? / 这是什么？

**Reframe Destiny** is an interactive website prototype that helps **young people aged 15–35** recognize and reframe **gendered narrative bias** in traditional BaZi (八字) and Western astrological readings.

**Reframe Destiny** 是一个交互式网站原型，帮助 **15–35 岁年轻人** 识别并重构传统八字与西方星盘解读中的 **性别叙事偏见**。
