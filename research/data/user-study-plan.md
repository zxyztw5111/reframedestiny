# User Study Plan

## Participants

15-25 young women, approximately ages 17-34, who have encountered BaZi, astrology, tarot, or similar personality/fate narratives.

## Procedure

1. Pre-survey: measure baseline ability to notice gendered bias in divination language.
2. MVP session: user completes one short Reframe Destiny journey.
3. Post-survey: repeat the key awareness/confidence questions.
4. Optional reflection: user writes one sentence about whether the reframed reading changed how they saw the original.
5. Optional short interview with 3-5 users if time allows.

## Draft Likert Items

Scale: 1 = strongly disagree, 5 = strongly agree.

1. I can recognize gender stereotypes in BaZi or astrology readings.
2. I notice when a reading treats marriage as the main measure of a woman's life.
3. I feel able to question a divination reading instead of accepting it as destiny.
4. I can rewrite a biased statement into a more empowering interpretation.
5. I think AI can help critique cultural narratives, not only generate them.

## MVP Data To Record

- Which system the user chooses: BaZi or astrology
- Whether the user completes the journey
- Which bias categories the user identifies
- One written reflection from the Court of Destiny step
- Post-session rating: "Did this help you notice bias?" from 1 to 5

## Ethics Notes

- Do not collect real birth data in the MVP.
- Explain that readings are simulated and educational.
- Store reflections anonymously for class research unless the participant gives permission to quote them.
