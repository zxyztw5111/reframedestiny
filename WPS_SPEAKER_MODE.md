# WPS 演讲者模式使用指南

> 适用：Mac 版 WPS 演示，Generation AI 期中 / Office Hour 汇报

---

## 方法一：演讲者视图（推荐）

### 1. 打开 PPT

- 双击桌面 **`Reframe-Destiny-Midterm.pptx`**
- 若提示选择软件，选 **WPS 演示**（WPS Presentation）

### 2. 进入演讲者视图

菜单栏：

```
幻灯片放映 → 演讲者视图
```

或英文版：

```
Slide Show → Presenter View
```

### 3. 你会看到

| 左边 / 中间 | 右边 |
|------------|------|
| 当前这页给观众看 | **下一张**预览 |
| | **演讲者备注**（你要读的话） |
| | 计时器 |

### 4. 翻页

- 键盘 **→** 或 **空格** = 下一页
- **←** = 上一页
- **Esc** = 退出

## 方法二：给每页加「演讲者备注」

如果演讲者视图找不到，可以先把讲稿贴进备注：

### 步骤

1. 打开 PPT，切到 **普通视图**（能看到左侧缩略图）
2. 点底部 **「备注」** 区域（若没有，菜单：**视图 → 备注**）
3. 打开 `midterm-script-simple.md`，**按页复制**对应段落
4. 粘贴到当前页的备注框
5. 9 页都贴完后保存

### 放映时看备注

- **幻灯片放映 → 演讲者视图** — 备注会显示在右侧
- 或：**幻灯片放映 → 从头开始** + 自己另开讲稿文件对照

## 方法三：分屏 / 手机看讲稿（最简单）

1. Mac 连接投影仪或共享屏幕 → 只共享 **PPT 全屏**
2. 手机上打开 `midterm-script-simple.md`（AirDrop 到手机，或用备忘录粘贴）
3. 眼睛看 PPT，低头看手机读对应段落

## 常见问题

**Q: WPS 没有「演讲者视图」？**  
A: 更新 WPS 到最新版；或用 **Keynote / PowerPoint** 打开同一文件，Keynote 按 **⌥ + ⌘ + P** 进入演讲者模式。

**Q: 字体太小？**  
A: 演讲者视图右侧备注可放大；或改用手机看简化讲稿。

**Q: 7 分钟不够？**  
A: 优先讲 Slide 1–3 和 7，Literature Review 可缩短到 30 秒。
