# Reframe Destiny — MVP Spec

## 它做什么

An interactive website where young women enter birth data (BaZi or astrology), see a traditional divination reading side-by-side with an AI-reframed version, and discover gendered narrative bias through guided interaction.

## 给谁用

Young women aged 17–34 who encounter traditional BaZi or astrology and want to question — not blindly accept — the stories these systems tell about their lives.

## 核心流程

1. User chooses a system: **BaZi** or **Western astrology**
2. User enters birth date, time, and place (simulated data in MVP)
3. System shows a **traditional reading** with gendered language
4. **Bias Scanner** highlights bias categories (marriage centrism, gender stereotypes, fatalism, etc.)
5. **AI Reframe** presents a rewritten, more equitable narrative
6. User collects **bias fragments** in a constellation map (progressive unlock across sessions)

## 不在 MVP 里

- Real AI API integration (uses curated simulated readings for now)
- Full 20-fragment game with all levels
- User accounts / cloud database
- Payment or social sharing
- Validating whether BaZi/astrology is scientifically true

## 怎么收集反馈

- Pre/post 5-item Likert survey (bias awareness before vs. after)
- Optional reflection text in "Court of Destiny" step
- Log: which bias fragments unlocked, journey completion rate (localStorage)
- Planned: 15–25 user tests before final paper (July)

## Research Question & Hypothesis

**Research Question:** How does an AI-assisted interactive website affect young women's ability to identify and reframe gendered narrative bias in traditional BaZi and Western astrological readings?

**Hypothesis:** I hypothesize that after using Reframe Destiny, young women will show increased awareness of gender bias in divination narratives and greater confidence in reframing these narratives for themselves.
